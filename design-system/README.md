# Design System

Use this folder for **future tasks**, **redesigns**, or **new pages**.

- **[DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md)** — Full design system: colors, typography, spacing, components, patterns, and checklist.
- **[design-system.json](./design-system.json)** — Machine-readable tokens (theme, breakpoints, component defaults). Sync with `src/styles/tokens.css` when you change brand or spacing.

Implementation lives in:
- `src/styles/tokens.css` — CSS variables
- `src/styles/components.css` — Button, Card, Badge, CodeBlock
- `src/styles/globals.css` — Utilities and base styles
